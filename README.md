## 📂 템플릿 종류

### 1. General (일반 대사관) ✅ **완성**
- **특징**: 공식적이고 신뢰감 있는 디자인
- **주요 기능**: 비자 안내, 영사 서비스, 문화 소개
- **완성도**: 100% (모리타니아 대사관 템플릿)
- **위치**: `/templates/embassy/general/`
- **다국어**: 한국어, 영어, 프랑스어

### 2. Consulate (영사관)
- **특징**: 대사관과 유사하되 영사업무 특화
- **주요 기능**: 여권 발급, 시민 서비스, 긴급 연락
- **색상 팔레트**: 네이비, 골드, 화이트
- **개발 우선순위**: 2순위

### 3. Government (정부기관)
- **특징**: 공공성과 접근성 강조
- **주요 기능**: 정책 안내, 민원 서비스, 공지사항
- **색상 팔레트**: 블루, 그린, 화이트
- **개발 우선순위**: 3순위

### 4. Public (공공단체)
- **특징**: 시민 친화적이고 투명한 디자인
- **주요 기능**: 사업 안내, 참여 프로그램, 소식
- **색상 팔레트**: 그린, 오렌지, 화이트
- **개발 우선순위**: 4순위

## 🚀 핵심 기능

### ✅ 완성된 기능 (General 템플릿)
- **다국어 지원**: 한/영/불 자동 전환
- **비자 정보**: 종류별 상세 안내
