# 🇲🇷 모리타니아 대사관 템플릿

*완성일: 2025-09-01*  
*상태: ✅ 완성*  
*품질: 90/100*

## 📋 템플릿 개요
- **국가**: 모리타니아 이슬람 공화국 (Islamic Republic of Mauritania)
- **수도**: 누악쇼트 (Nouakchott)
- **언어**: 아랍어, 프랑스어
- **특징**: 다국어 지원 (한/영/불), Excel 자동화

## 🌟 주요 기능
- 다국어 웹사이트 (한국어/영어/프랑스어)
- Excel 기반 콘텐츠 자동 업데이트
- 영사 업무 안내 시스템
- 양국 관계 정보 페이지
- 공지사항 및 뉴스 관리

## 🎨 디자인 특징
- 모리타니아 국기 색상 (녹색, 노란색) 활용
- 이슬람 문화 고려한 정중한 디자인
- 아랍어 RTL 레이아웃 지원 가능

## 🔧 기술 스택
- Next.js 15.5.2 + TypeScript
- next-intl (다국어)
- Tailwind CSS
- Excel 자동화 시스템

## 📞 활용 용도
- 다른 아프리카 국가 대사관
- 이슬람 국가 대사관
- 다국어 지원 공공기관
